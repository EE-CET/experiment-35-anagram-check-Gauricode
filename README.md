# Experiment 35: Anagram Check

## Problem Statement

Check if two strings are anagrams.
Two strings are anagrams if they contain the same characters with the same frequencies, ignoring the order.

**Note:** The check should be **case-insensitive**.

## Input Format

* Two strings on separate lines.

## Output Format

* Print `true` if they are anagrams, otherwise print `false`.

### Example 1

**Input:**

```text
Triangle
Integral
```

**Output:**

```text
true
```

### Explanation
Both words contain the same letters (a, e, g, i, l, n, r, t) when case is ignored.
